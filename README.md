🌍 A Global Gap in Access to Life Saving Treatment

This project was created for the Data Analytics module at Dublin City University (DCU).

The website explores global inequalities in access to oral rehydration salts (ORS) and zinc treatment for children under five with diarrhoea. Using UNICEF and World Bank data, it examines global treatment coverage, whether wealth improves access to treatment, and how coverage has changed over time.

The project was developed in Google Colab using Python, Polars, Plotnine, GeoPandas, and Quarto, then published through GitHub Pages.
